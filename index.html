notebooklm-py리서치를 Google NotebookLM에 아웃소싱

딥리서치로 컨텍스트를 태우지 말고,
조사는 NotebookLM이 하고
Claude는 결과만 받아옵니다.
출처·팟캐스트·퀴즈까지 전부 명령어로.
⭐ 17.5k · Python CLI + AI 에이전트 연동
github.com/teng-lin/notebooklm-py

📦 설치 (3단계)

터미널
1. 설치 — 위 준비 단계에서 uv를 먼저 설치했다면 그대로 됩니다 (따옴표까지 그대로 복사하세요)
uv tool install "notebooklm-py[browser]"
터미널
2. 구글 로그인 — 입력하면 브라우저 창이 열립니다. 평소 NotebookLM 쓰는 구글 계정으로 로그인하세요
notebooklm login
터미널
3. 잘 연결됐는지 확인 — OK가 나오면 성공
notebooklm auth check --test

▶️ 사용 흐름 (예시)

터미널 
notebooklm create "내 리서치"                # 노트북 만들기
notebooklm source add "https://example.com"  # 자료 넣기
notebooklm ask "핵심 주제가 뭐야?"            # 질문하기
notebooklm generate audio --wait             # 팟캐스트 생성
⚠️ 구글 비공식 API를 사용합니다. 예고 없이 바뀔 수 있으니 개인 프로젝트 용도로 추천. Claude Code에 연결하면 Claude가 위 명령을 대신 실행해 줍니다.

graphify코드·문서를 기억하는 지식 그래프

Claude는 세션마다 코드베이스를
처음부터 다시 검색합니다.
graphify는 코드, 문서, PDF, 영상까지
쿼리 가능한 지식 그래프로 바꿔줍니다.
⭐ 81.5k · 약 40개 언어 지원 · Claude Code 등 20+ 도구 호환
github.com/Graphify-Labs/graphify

📦 설치 (2단계)

터미널
1. 도구 설치 — 철자 주의! 패키지 이름은 y가 두 개인 graphifyy 입니다
uv tool install graphifyy
터미널
2. Claude Code에 스킬 등록 — 자동으로 연결해 줍니다
graphify install

▶️ 사용법

Claude Code
분석하고 싶은 프로젝트 폴더에서 claude 실행 후, 채팅창에 입력 (마침표는 "지금 이 폴더"라는 뜻)
/graphify .
💡 완료되면 파일 3개가 생깁니다 — graph.html(더블클릭하면 그래프 시각화) · GRAPH_REPORT.md(요약 보고서) · graph.json(쿼리용 데이터)

impeccable'AI가 만든 티' 지우는 디자인 시스템

보라색 그라데이션, 카드 속 카드 —
다 비슷한 AI 화면에서 탈출.
디자인 명령 23종과 안티패턴 감지 46종으로
Claude가 템플릿이 아니라 감각으로 디자인합니다.
⭐ 45.2k · Claude Code / Cursor / Copilot 등 지원
github.com/pbakaus/impeccable

📦 설치 (3단계)

터미널
1. 먼저 내 프로젝트 폴더로 이동합니다 (cd = 폴더 이동 명령)
cd 내프로젝트폴더경로   # 예: cd D:\my-website
터미널
2. 설치 실행 — Node.js가 있어야 합니다 (준비 ① 참고). 중간에 질문이 나오면 엔터로 진행
npx impeccable install
Claude Code
3. 같은 폴더에서 claude 실행 후 최초 세팅 한 번
/impeccable init

▶️ 자주 쓰는 명령

Claude Code 
/impeccable audit     # 지금 디자인 문제점 점검
/impeccable polish    # 디테일 다듬기
/impeccable critique  # 신랄한 디자인 비평 받기
